## Project Summary
This project, "Celo Composer - Simple DeFi AI Agent Template," aims to provide an AI agent capable of performing DeFi operations on the Celo network, specifically utilizing Uniswap V3 for token swaps and quote generation. The AI agent offers a natural language interface for users to interact with DeFi protocols.

## Architecture and Structure
-   **Overall project structure**: The project is structured as a Node.js application with a `plugin` directory containing the core DeFi logic. `index.ts` serves as the main entry point, setting up the AI agent and the interactive CLI.
-   **Key components and their Self interactions**:
    -   `index.ts`: Initializes the `viem` wallet client and integrates `uniswap` as an on-chain tool via `@goat-sdk`. It handles user prompts and AI responses.
    -   `plugin/uniswap.service.ts`: Contains the `UniswapService` class, which implements the `getQuote` and `executeSwap` tools for the AI agent, interacting with Uniswap V3 contracts on Celo.
    -   `plugin/constants.ts`: Defines Celo token addresses, Uniswap V3 contract addresses, and ABIs.
    -   `plugin/parameters.ts`: Defines Zod schemas for the parameters of the AI agent's tools.
-   **Smart contract architecture (Self-related contracts)**: No Self-related smart contracts are identified. The project interacts with standard Uniswap V3 contracts (Factory, QuoterV2, SwapRouterV2) and ERC20 tokens.
